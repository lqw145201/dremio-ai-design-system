2 — Semantic Token Usage

Token Hex Use ⚠️ Do NOT use for
--foreground #202124 Primary text, titles, active nav labels —
--secondary-foreground #505862 Table headers, stat labels, section descriptions Placeholders or disabled text
--muted-foreground #B0B7BF Placeholders, disabled text, empty states Everything else — do not use as body text
--muted #EEEFF1 Dividers, muted backgrounds, chip backgrounds —
--accent #008489 Links, ghost button text, active nav label (expanded) Code syntax colors — use syntax palette instead
--primary #43B8C9 Primary button bg, active state highlights Code syntax colors
